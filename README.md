# zh-portal-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容不针对任何特定域名或网站，仅作为通用型页面集合进行存储与展示。

## 项目简介

zh-portal-journal 是一个以 HTML 页面为主要内容的归档仓库。仓库中的每个页面均为独立文件，不依赖外部数据库或后端服务。适合用于存放静态资讯、存档页面或临时展示内容。

## 目录说明

```
.
├── index.html          # 入口页面（如有）
├── pages/              # 存放主要 HTML 页面
├── assets/             # 样式、脚本、图片等资源文件
└── README.md           # 本文件
```

- `pages/`：各独立 HTML 文件，命名清晰，便于索引。
- `assets/`：页面引用的公共资源，按类型组织。

## 页面归档说明

- 每个 HTML 页面均为独立文档，可直接通过浏览器打开。
- 页面之间无强制关联，可单独更新或移除。
- 归档内容不涉及特定域名或网站，仅作为通用资料存放。

## 维护说明

- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并确保资源引用路径正确。
- 修改或删除页面时，请同步更新相关引用。
- 欢迎通过 Pull Request 提交改进或修正。

## 许可

本仓库内容仅供学习与归档使用。如有特殊用途，请自行判断合规性。
